## What is Gyroflow

**Gyroflow** is a cross-platform post-stabilization tool that aligns **gyro data** with footage to compute accurate transforms and compensate for shake. It ingests IMU tracks from action cameras and flight controllers (GoPro/DJI/Sony/Insta360, Betaflight/INAV Blackbox), maintaining natural motion, controlled crop, and reliable horizon lock for FPV, action, inspection, and cinematic workflows.
